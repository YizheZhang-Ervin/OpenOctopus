# Skills

This directory contains skills that extend Octopus's capabilities.

## Structure

Each skill is a directory containing a `SKILL.md` file that describes how to use the skill.

## Adding Custom Skills

To add a custom skill:

1. Create a new directory in your workspace's `skills` folder
2. Add a `SKILL.md` file with the skill description
3. Optionally add any scripts or supporting files

## Built-in Skills

- **memory**: Manage long-term memory
- **weather**: Get weather information
- **summarize**: Summarize text
- **github**: Interact with GitHub repositories
- **tmux**: Manage tmux sessions
- **cron**: Schedule tasks
- **clawhub**: Interact with ClawHub
- **skill-creator**: Create new skills
